# Palantir Core Concepts

## 来源信息

- **来源**: Palantir Foundry 官方文档
- **URL**: https://palantir.com/docs/foundry/ontology/core-concepts/

## 一句话摘要

Ontology 核心概念的综合文档页面，覆盖了对象类型、链接类型、动作类型、函数、接口、搜索、场景等全部核心概念。

## 解决的问题

为 Ontology 所有核心概念提供一个集中的参考入口。

## 关键结构

文档包含以下主要章节：
- Ontologies（本体定义、分支方案、共享、迁移、使用量）
- Object and link types（对象类型、链接类型、属性、结构类型、值类型）
- Action types（动作类型定义、参数、函数支持、副作用、权限、监控）
- Functions（函数定义、管理、消费、多语言支持、单元测试）
- Interfaces（接口定义、实现、扩展、约束）
- Ontology search（语义搜索、文档处理、Ontology Augmented Generation）
- Ontology scenarios（场景概览、临时/持久化场景、合并）
- Applications（Object Explorer、Object Views、Vertex、Map 等应用）
