# Wiki — 知识编译层

LLM 维护的结构化 Wiki，所有知识页存放在此。

## 页面类型

| 目录 | 用途 |
|------|------|
| `资料/` | 单篇来源的摘要和定位（论文、文章、书摘等） |
| `实体/` | 人物、组织、产品、项目、工具等 |
| `概念/` | 方法、模式、理论、框架等 |
| `综合/` | 跨来源形成的判断、综述、主题地图等 |
| `对比/` | 区分容易混淆的概念、产品、方案等 |
| `查询/` | 有价值的问答归档 |

## 约定

- 目录名和文件名尽可能用中文
- 术语型专名可保留英文，如 `AI Agent.md`
- 使用 Obsidian wikilink：`[[Knowledge/wiki/概念/xxx]]`
- 页面添加 frontmatter 标注类型和状态
